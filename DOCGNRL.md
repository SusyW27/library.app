# library.app
Sistema web para gestionar una biblioteca sencillo
Nombre del Proyecto: Sistema Básico de Gestión de Biblioteca
Versión: 1.0.0
Objetivo: Proporcionar una interfaz web sencilla (CRUD) para gestionar Autores, Libros y registrar Préstamos realizados a Usuarios.
Tecnologías:PHP 7+ (Lógica de Negocio), HTML5 (Vistas), MySQL/MariaDB (Base de Datos), Estilo de Programación Procedural (con separación básica de archivos).
Base de Datos :Diseño Relacional en 3FN (Tercera Forma Normal) para escalabilidad y mantenimiento.
